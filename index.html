Automation Solution Proposal for Aladdin Metal Products

Executive Summary

This document outlines the technical and business specifications for automating key processes at Aladdin Metal Products, aiming to enhance efficiency, reduce manual errors, and provide substantial cost savings. The goal is to achieve a revenue target of $121,800 through a strategic pricing model based on 6-10 licenses and to generate long-term value including employee raises and funding for a new building within four years.

Business Goals

  • 1 Increase Efficiency: Automate repetitive tasks to allow employees to focus on higher-value activities.
  • 2 Reduce Costs: Minimize manual errors and reduce labor costs associated with repetitive tasks.
  • 3 Enhance Data Accuracy: Improve data accuracy through automated data entry and processing.
  • 4 Generate Revenue: Monetize the automation software by selling licenses, aiming to generate $121,800.
  • 5 Support Employee Growth: Enable raises for peers through productivity gains and cost savings.
  • 6 Fund Strategic Investments: Generate sufficient savings and revenue to invest in a new building within four years.

Project Overview

This project aims to automate interactions with several critical applications and processes at Aladdin Metal Products, including SolidWorks 2022+, Excel, Epicor Eclipse Eterm, Realtrac Shop Management Software, Microsoft file system, PDFs, Google Chrome, Microsoft Printer, and Microsoft local server folders.

Technical Requirements

  • SolidWorks 2022+ Automate design tasks and data extraction using SolidWorks API.
  • Excel Automate data manipulation and reporting.
  • Epicor Eclipse Eterm Automate terminal interactions.
  • Realtrac Shop Management Software Automate GUI interactions.
  • Microsoft File System Automate file management tasks.
  • PDFs Automate PDF creation, manipulation, and data extraction.
  • Google Chrome Automate browser tasks for web-based processes.
  • Microsoft Printer Automate printing tasks.
  • Microsoft Local Server Folders Automate server folder management.

Technology Stack and Libraries

  • Python Primary programming language for scripting and automation.
  • Libraries:
    • SolidWorks `swpy`
    • Excel `pandas`, `openpyxl`, `xlrd`, `xlwt`
    • Epicor Eclipse Eterm `pyte`, `pyexpect`
    • Realtrac `pyautogui`, `pywinauto`
    • File System `os`, `shutil`, `pathlib`
    • PDFs `PyPDF2`, `reportlab`, `pdfplumber`
    • Google Chrome `selenium`, `beautifulsoup4`, `requests`
    • Printer `win32print`, `win32api`

Implementation Plan

- Conduct detailed analysis of current processes and identify key areas for automation. - Engage with CAD Lead, Sales Manager, GM, Plant Manager, and Owners to gather requirements.

- Develop automation scripts for each application and process. - Implement a licensing system with key validation and time-limited licenses. - Obfuscate and encrypt code to protect intellectual property.

- Perform unit testing and integration testing. - Conduct user acceptance testing (UAT) with key stakeholders.

- Deploy automation tools across the company. - Provide training and documentation to users.

- Offer ongoing support and updates. - Gather feedback and continuously improve the automation tools.

Setting Up Your Python Environment and Project

Here's a complete guide to setting up your Python environment and project, incorporating best practices such as using virtual environments, version control with Git, and structuring your project.
1. Install Python and Pip
Ensure you have Python 3.8+ installed on your system. You can download it from Python's official website.
2. Set Up a Virtual Environment

  1. Create a Project Directory: 
    mkdir AMP_Automation_Project
cd AMP_Automation_Project
  2. Set Up Virtual Environment: 
    python -m venv venv
  3. Activate Virtual Environment: 
    # On Windows:
venv\Scripts\activate
# On macOS/Linux:
source venv/bin/activate
3. Install Necessary Libraries
Install the libraries you’ll need for your project: 
pip install swpy pandas openpyxl xlrd xlwt pyte pexpect pyautogui pywinauto os shutil pathlib PyPDF2 reportlab pdfplumber selenium beautifulsoup4 requests win32print pywin32
4. Set Up Version Control with Git
  1. Initialize Git Repository: 
    git init

    venv/
__pycache__/

*.pyc
*.pyo
*.pyd
.vscode/
  3. Initial Commit: 
    git add .
git commit -m "Initial commit"
